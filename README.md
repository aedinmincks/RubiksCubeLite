# RubiksCubeLite

##目前仅支持以下几种魔方

 2： 2阶魔方  
 3： 3阶魔方  

## example

```shell
> .\RubiksCubeLite.exe 3
#get
[UF,UR,UB,UL,DF,DR,DB,DL,FR,FL,BR,BL,UFR,URB,UBL,ULF,DRF,DFL,DLB,DBR]
#set [UF,BR,DR,DB,DL,FR,UB,UL,UR,DF,BL,FL,RBU,DBR,DLB,FDR,RUF,FLD,ULF,UBL]
success
#show
   DDD
   DUB
   RUB
BBFDFURRBRRL
LLFDFURRLBBF
LLFDLURRLBBF
   LDF
   UDF
   UUU
#solve
L2B2R'R2D'
#exit
Good bye!
```
